# Call Taxi Booking System
This is a console-based Call Taxi Booking System developed in Java using Object-Oriented Programming (OOP) principles and clean layered architecture.

The system simulates real-world taxi booking operations including customer booking, taxi allocation, trip calculation, and earnings tracking.

This project was built as part of Low-Level Design (LLD) practice to understand class design, object interaction, and scalable architecture.

## Features Implemented

Multiple taxis available in the system

Taxi allocation based on:

Nearest available taxi

Minimum earnings logic

Customer booking with:

Pickup point

Drop point

Pickup time
